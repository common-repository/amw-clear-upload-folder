Removes unused files from the 'uploads' folder

== Description ==

<strong>
Before using this plugin, please make sure that you BACKUP 'upload' folder. This is the most important step!
Also do not forget Rescan Images every time you change settings
</strong>
<strong>
Please note that 'max_execution_time' PHP setting should be minimum 90 seconds for the correct script work.
</strong>

AMW Clear Upload Folder will compare images entries from the database with filenames in upload directory.
All files that are missing in the database will be removed from the filesystem.

<h3>HOW TO USE</h3>

1. Install and activate plugin
2. Go to the 'Settings' tab and choose needed settings. Please note everytime you make changes to the settings,
you need to rescan images on the 'List of files' tab
3. Go to 'List of folders' tab under plugin page and choose folders that you DO NOT WANT to clean
4. After this you can navigate to 'Main' tab and click 'Run cleaner' button
5. That's all plugin will start cleaning files

== Installation ==

1. Upload the entire amw-clear-upload-folder folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will find 'AMW Clear Upload Folder' under 'Tools' in your WordPress admin panel.

= Installation Instructions =

<h3>UPLOADING IN WORDPRESS DASHBOARD</h3>

1. First download the ZIP file from WordPress website
2. Log in to your website administrator panel
3. Go to the ‘Add New’ in the plugins dashboard, click “Upload Plugin”
4. Upload AMW Clear Upload Folder ZIP file by choosing it from your computer
5. Click Install Now button
6. Then click Activate Plugin button
7. You can find 'AMW Clear Upload Folder' under 'Tools' in your WordPress admin panel

<h3>USING THE WORDPRESS PLUGINS DASHBOARD</h3>

1. Go to the ‘Add New’ in the plugins dashboard
2. Search for ‘AMW Clear Upload Folder’
3. Click Install Now button
